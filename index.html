Pencil

Pencil is an answering engine, a command line tool based on chemistry.

Download Here

Sort

Sort command is use to sort the series of elements on the basis of given property. Any Series of elements e,g : 3d,2p can be sorted.

Plot

We incorporated (C)GCC Graph to plot values of property of any series in periodic table e.g : 6f,6p.

Get

Using get command and along with name of any series name e,g: alkali, 7f and D we can get all the property value of the all the element of that series in tabular form.

One Element

Using get command and along with its period, group, and f-flag we can get all the property of one elements.

State

Using state along with get command with specified temperature we can know about the state of the element at, above or below of given temperature.

Set

Using set command we can update the any property value of any element.
